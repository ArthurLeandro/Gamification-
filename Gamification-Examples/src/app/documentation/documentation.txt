# Educa

Educa é um portal voltado para o professor planejar AULAS e PLANOS DE AULA. 

## Exemplos de Gamificação

Educa contará com uma vasta gama de técnicas de gamificação, essas gamificações podem ser acessadas clicando
[aqui] (https://github.com/ArthurLeandro/Gamification-.git).

## On Boarding

On Boarding é todo um grupo de técnicas que têm um caráter introdutório e/ou customizável. Iremos implementar 
três técnicas no On Boarding:

  * Help Box 
  * Tutorial
  * Customização


### Help Box

São diversos botões colocados dentro de um card de ATIVIDADE ou AULAS que o usuário ao clicar terá um "modal"
aberto em sua frente, e esse modal conterá informações sobre o campo em que o usuário acabou de clicar.

### Tutorial

É uma opção que o usuário poderá marcar sempre que ele precisar de uma ajuda para circular entre as opções de
campos e funcionalidades do EDUCA.

### Customização

É uma aba que permitirá que o usuário customize as cores, coloque adornos em sua foto, personalize seu título 
dentro do site.

## Pontuação

É um conjunto de maneiras de se metrificar a experiência do usuário.As experiências são:
  
  *Educoin
  *Karma
  *Pontos
  *Reputação

  ### Educoin

  Educoins são moedas que o usuário recebe ao concluir uma missão, ele usará essas moedas
  para desbloquear RECURSOS, ou ATIVIDADES criadas pela própria equipe desenvolvedora da
  plataforma.

  ### Karma

  Karma é um multiplicador que o usuário recebe sempre que ele conclui alguma ação que o 
  beneficia com pontos. E caso esse usuário tenha seu conteúdo apontado como fraude o Karma
  entrará em ação e irá tirar boa parte de seus pontos.

  ### Pontos

  Pontos são pontos dado ao usuário ao concluir missões, ao efetuar login, ao postar conteúdo
  ao mandar mensagem para outros usuários, ao avaliar o conteúdo. Esses pontos serão usados 
  em um ranking anual que recompensará os melhores rankeados com premiação ainda a definir, 
  além de claro definir o nível do usuário.

  ### Reputação

  Reputação são pontos dados ao conteúdo do usuário e são usados como filtro de qualidade.
  Essa reputação é dado como favorito, como nota de 1 a 5 estrelas.  

## Nível

É uma forma de avaliar o engajamento do usuário com o sistema. Faz uso extensivo dos pontos
adquiridos pelo usuário para quantificar seu nível. Esse nível desbloqueia novos "assets"
para que o usuário possa customizar seu Dashboard.

## Missões

É um sistema desenvolvido para que o usuário se mantenha engajado por mais tempo com a 
plataforma. Esse sistema é baseado em ações que o usuário necessita de fazer para receber 
pontuação. As missões conssistem em:

  * Missões Sazonais
  * Missões Comuns
  * Missões Cooperativas

  ### Missões Sazonais

  São missões promovidas pelo EDUCA em que o usuário utilize o apoio de sua sala para efetuá-la
  com sucesso, essas missões recebem prêmios extras, e fazem uso extensivo da funcionalidade de 
  fazer upload de EVIDÊNCIAS.

  ### Missões Comuns

  São missões comuns já estabelecidas pelo próprio sistema onde o usuário opera as funcionalidades
  já desenvolvidas pelo sistema, planejar um PLANO DE AULA ou cadastrar um RECURSO por exemplo.  

  ### Missões Cooperativas

  São missões onde o usuário utiliza a funcionalidade de cooperação com outro usuário para
  criar um PLANO DE AULA com co-autores.

## Medalhas

É um sistema onde o usuário recebe MEDALHAS por suas conquistas, como por exemplo postar
100 aulas.
