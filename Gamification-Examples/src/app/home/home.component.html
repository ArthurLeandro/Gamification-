On Boarding

On Boarding é todo um grupo de técnicas que têm um caráter introdutório e/ou customizável.

Pontuação

É um conjunto de maneiras de se metrificar a experiência do usuário.

Nível

Faz uso extensivo dos pontos adquiridos pelo usuário para quantificar seu nível.

Missões

É um sistema desenvolvido para que o usuário se mantenha engajado por mais tempo com a plataforma.

Medalhas

Sistema que recompensa o usuário por engajar com a plataformaEduca
