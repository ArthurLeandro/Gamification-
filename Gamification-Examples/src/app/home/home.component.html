Interação
A interação é o encapsulamento de diversas técnicas do ramo de Gamificação determinada para este projeto.Tem o intuito de fazer com que o usuário continue engajado e ativo dentro do sistema durante mais tempo, permitindo que o tenha uma maior duração. As técnicas são: Nível, Pontuação, Botão de Ajuda, Barra de Progresso,Educoin, Pontos de Karma. Interação


Eventos
Os eventos tratam de interações que o usuário possuem com o sistema. Tem o intuito de despertar no usuário "uma potência para ação" recompensando o usuário caso ele consiga concluir. As técnicas implementadas são Missões Comuns, , Missões Cooperativas e Medalhas. Eventos


Customização
Área onde o usuário pode customizar pequenos componentes do sistema para deixar a experiência de cada usuário única. Customização


Tutorial
Pequena experiência que auxilia o usuário a ter uma noção das funcionalidades do sistema. Tutorial


Documentação
Documentação explicando de maneira mais profunda as funcionalidades e pequenos componentes que compõem o sistema. Documentação
